Geçerli kaynak kodlarının çalıştırılması için MacOSX işletim sisteminde kaynak kodlarının bulunduğu dosya konumuna gelindikten sonra terminalde "make" komutunun çalıştırılıp ardından "./run" komutunun çalıştırılması yeterlidir.


Program çalıştıktan sonra rotate için:

"w,a,s,d,q,z" tuşları kullanılmalıdır.

Scale için:

"1,2" tuşları kullanılmalıdır.

Translate için:

"t,f,g,h,r,v" tuşları kullanılmalıdır.

Çıkmak için:

"ESC" tuşu kullanılmalıdır.
